    @mixin headings($1: default none, $2: default none, $3: default none, $4: default none, $5: default none, $6: default none) { ... }
    View source

    Description

    Dynamically create styles for headings.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $1

    applies styles for h1

    		Typedefault none
    $2

    applies styles for h2

    		Typedefault none
    $3

    applies styles for h3

    		Typedefault none
    $4

    applies styles for h4

    		Typedefault none
    $5

    applies styles for h5

    		Typedefault none
    $6

    applies styles for h6

    		Typedefault none

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    Inclusive Usage Sass

    @include headings() {
    css: here;
}

    Inclusive Usage CSS Output

    h1, h2, h3, h4, h5, h6 {
    css: here;
}

    Specific Usage Sass

    @include headings(1) {
    css: here;
}

    Specific Usage CSS Output

    h1 {
    css: here;
}

    • Sean Kelly
